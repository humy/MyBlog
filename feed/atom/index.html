Enter file in which to save the key (/home/you/.ssh/id_rsa):</code></pre> <p>直接按Enter就行。然后，会提示你输入密码，建议输一个。</p> <pre><code>Enter same passphrase again: [Type passphrase again]</code></pre> <p>完了之后，大概是这样：</p> <pre><code>Your public key has been saved in /home/you/.ssh/id_rsa.pub.
